pub struct ControlTimer<T> { /* fields omitted */ }

ControlTimer is used to work with a TCC peripheral instance.

All configuration options that are not configurable when the instance is enabled are found in the TimerConfig<TCC> type. You can use one of the tccx methods on TimerConfig<TCC> to configure a new instance of ControlTimer, or use the reconfigure method on ControlTimer to reconfigure an existing timer instance.

Timer Width

The size of all timer value accesses are always 32-bit, even if the timer instance is 24-bit or 16-bit wide. As per the reference manual, the upper bytes will be read as zero.
